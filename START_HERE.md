# 👋 Welcome! Start Here

Welcome to the **Telegram Anonymous Messages Bot** project!

This is a complete, production-ready bot that lets users send and receive anonymous messages via personal links.

---

## ⚡ Quick Start (5 Minutes)

### 1️⃣ Run Installer
```bash
# Windows
install.bat

# Mac/Linux
bash install.sh
```

### 2️⃣ Get Bot Token
- Open Telegram → Search **@BotFather**
- Send: `/newbot`
- Copy your token

### 3️⃣ Configure & Run
```bash
# Edit .env file and paste your token
python main.py
```

### 4️⃣ Test in Telegram
- Find your bot
- Click `/start`
- Share your personal link!

**Done! 🎉 Bot is running.**
